# Portfolio - Russell Rojas

Ce dépôt contient mon site web personnel, présentant mes compétences, mes projets et mes informations de contact.

## Technologies utilisées
- HTML5, CSS3, JavaScript
- Design responsive

## Sections
- À propos (About)
- Compétences (Skills)
- Projets (Projects)
- Contact

## Comment utiliser

- Pour visualiser le site localement, ouvrir le fichier `index.html` dans un navigateur.

- Pour le déployer avec **GitHub Pages** :
  1. Aller dans les paramètres du dépôt → **Pages**
  2. Sélectionner la branche `main` et le dossier `/` (root)
  3. Cliquer sur “Save”
  4. Le site sera alors disponible à l'adresse : `https://russellrojas.github.io`


## Structure du projet

- `index.html` → page principale
- `css/` → fichiers de style
- `js/` → fichiers JavaScript
- `images/` → captures, photos, icônes
- `assets/` → autres fichiers (optionnel)
